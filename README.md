# Star War游戏设计
## 设计思路
#### 通过传感器控制飞机的移动，飞机可以发射炮弹
#### 当炮弹和敌机之间的距离小于设定值时敌机消失
## 设计要点
#### 控制敌机随机生成
#### 通过传感器控制飞机
####  控制敌机消失
